- This folder contains compress folder with JCAMP-DX files for individual molecules and metadata.json file.
- Data for individual molecules like canonical smiles and filenames can be found in metadata.json
- Jcamp files are exported from Chemotion Repository
- Individual files of the molecules follow the format of data storage of chemotion repository. Each molecule can have multiple samples depending on the publisher.
- Jcamp file names can be mapped to their respective meta data in meta_data.json via 'identifier' tag in the json file.
